AI & Machine Learning Notes

Introduction to AI and Machine Learning

Artificial Intelligence (AI) refers to the simulation of human intelligence in machines that are designed to think and learn like humans. Machine Learning (ML) is a subset of AI that involves the use of algorithms and statistical models to enable computers to improve their performance on a task through experience.

Types of Machine Learning

  • Supervised Learning: The model is trained on labeled data, where the desired output is known. Examples include classification and regression tasks.
  • Unsupervised Learning: The model is trained on unlabeled data and must find patterns and relationships in the data. Examples include clustering and dimensionality reduction.
  • Reinforcement Learning: The model learns by interacting with its environment and receiving feedback in the form of rewards or penalties. Examples include game playing and robotics.

Key Concepts in Machine Learning

Features and Labels

Features are the input variables used by the model to make predictions, while labels are the output variables the model aims to predict.

Training and Testing

Training involves fitting a model to a dataset, while testing evaluates the model's performance on unseen data.

Overfitting and Underfitting

Overfitting occurs when a model learns the training data too well, including its noise, while underfitting occurs when a model is too simple to capture the underlying patterns in the data.

Cross-Validation

A technique used to assess the performance of a model by splitting the data into multiple subsets and training/testing the model on different subsets.

Popular Algorithms and Models

  • Linear Regression: A regression model that predicts a continuous output based on the linear relationship between input features.
  • Decision Trees: A model that makes decisions based on a series of questions about input features.
  • Support Vector Machines (SVM): A classification model that finds the optimal hyperplane to separate classes in the feature space.
  • Neural Networks: Models inspired by the human brain, consisting of interconnected nodes (neurons) that process data in layers.
  • K-Nearest Neighbors (KNN): A classification or regression model that makes predictions based on the closest training examples in the feature space.
  • Clustering Algorithms: Algorithms like K-Means and Hierarchical Clustering that group similar data points together.

Deep Learning

Deep Learning is a subset of machine learning that uses neural networks with many layers (deep neural networks) to model complex patterns in large datasets. It is particularly effective in tasks such as image and speech recognition.
