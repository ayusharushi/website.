Operating System Notes

Introduction to Operating Systems

An operating system (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs. It acts as an intermediary between users and the computer hardware.

Key Functions of an Operating System

  • Process Management: Handles the creation, scheduling, and termination of processes. Ensures efficient execution of processes.
  • Memory Management: Manages the computer's memory, including RAM and cache. Allocates memory to processes and handles memory deallocation.
  • File System Management: Manages files on the disk, including file creation, deletion, and access control.
  • Device Management: Manages device communication via drivers and ensures that hardware devices are properly utilized.
  • User Interface: Provides a user interface, which can be command-line (CLI) or graphical (GUI), allowing users to interact with the computer.

Types of Operating Systems

  • Batch Operating Systems: Processes jobs in batches without user interaction. Examples include early mainframe systems.
  • Time-Sharing Operating Systems: Allows multiple users to interact with the computer simultaneously. Examples include UNIX and Linux.
  • Distributed Operating Systems: Manages a group of separate computers as a unified system. Examples include Googleâ€™s Android and iOS.
  • Real-Time Operating Systems: Guarantees processing within a certain time constraint. Examples include embedded systems used in medical devices and automotive systems.
  • Network Operating Systems: Manages network resources and enables communication between devices on a network. Examples include Windows Server and Linux distributions for servers.

Process Management

Process management involves creating and managing processes. Each process has a process control block (PCB) that includes process ID, state, program counter, and other important information. Key concepts include process scheduling, context switching, and process synchronization.

Memory Management

Memory management involves managing the computer's memory, including allocation and deallocation of memory to various processes. Techniques include paging, segmentation, and virtual memory.

File System

The file system manages files and directories on storage devices. It handles file creation, deletion, and access control. Common file systems include FAT32, NTFS, and ext4.

Device Management

Device management includes managing hardware devices such as printers, disk drives, and network interfaces. It involves using device drivers to enable communication between the OS and hardware.

Security and Protection

Operating systems implement security measures to protect data and resources. This includes user authentication, access control, encryption, and auditing.

Popular Operating Systems

  • Windows: Developed by Microsoft, widely used in personal and business environments.
  • Linux: An open-source OS with various distributions like Ubuntu, Fedora, and Debian. Popular for servers and development environments.
  • macOS: Developed by Apple, used in Macintosh computers. Known for its user-friendly interface and integration with other Apple products.
  • Android: A mobile OS developed by Google, widely used in smartphones and tablets.
  • iOS: Developed by Apple, used in iPhones and iPads.
